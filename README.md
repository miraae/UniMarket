* 사용한 언어 : JAVA, XML, PHP 
* 사용한 시스템 : 안드로이드 스튜디오, Firebase, MYSQL, SourceTree
* 관련설명 : 구인과 구직 정보를 동시에 확인할 수 있는 메인화면을 통해서 대학생들이
사소하지만 사소하지 않은 고민거리를 해결하는 앱이다.
* 수상 : 한국정보과학회 회장상을 수상
* 벤치마킹 : 에브리타임과 당근마켓을 벤치마킹 하였다.
